# QRCodeFor1922 自製實聯制掃描器 For Android


官方版本請至[Play商店][1]下載

Android 6.0 以前請下載[此版][2]

暗黑版本(自動傳送簡訊)請至 [Release][3] 下載

## Library
此APP的QRCode解碼功能使用 [zxing-android-embedded][4] 的開放原始碼項目

5.0以後改用 [Google ML Kit][6]
